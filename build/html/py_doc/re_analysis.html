5. Python正则表达式分析利器¶

tags： Python 正则表达式

Note

在学习正则表达式及Python re模块中，发现一个分析正则模式的利器，下面根据自己在阅读代码时见到的最复杂的正则表达式，来进行分析。

import re
# 该正则表达式来源于bottle框架的URL正则匹配模式，非常复杂！
re.compile('(\\\\*)'\
        '(?:(?::([a-zA-Z_][a-zA-Z_0-9]*)?()(?:#(.*?)#)?)'\
          '|(?:<([a-zA-Z_][a-zA-Z_0-9]*)?(?::([a-zA-Z_]*)'\
            '(?::((?:\\\\.|[^\\\\>]+)+)?)?)?>))', re.DEBUG)
